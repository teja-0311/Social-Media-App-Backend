# Social Media App Backend

A backend social media application built using Node.js, Express.js, MongoDB, and EJS.

## Features

- User Authentication (JWT)
- Password Hashing (bcrypt)
- Create, Edit, and Like Posts
- Upload Profile Pictures (Multer)
- Protected Routes
- MongoDB Database

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- JWT
- bcrypt
- Multer

## Installation

```bash
git clone <repo-url>
cd social-media-app
npm install
```

Create a `.env` file.

```env
MONGODB_URI=
JWT_SECRET=
```

Run:

```bash
npm start
```
